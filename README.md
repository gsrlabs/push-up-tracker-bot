# 💪 Push-Up Tracker Bot

link: https://t.me/PushUpTracker_bot

Telegram-бот для учёта ежедневных отжиманий с персонализированной системой расчёта нагрузки, историей прогресса и рейтингом пользователей.

Проект реализован как production-ориентированное Go-приложение с PostgreSQL, Docker, CI и тестированием.

---

## 🚀 Возможности

### 🏠 Главное меню

* ➕ **Добавить отжимания** — ввод выполненного количества с моментальным обновлением прогресса
* ⚙️ **Дополнительно** — доступ к настройкам и статистике

---

### ⚙️ Дополнительные функции

* 🎯 **Тест максимальных отжиманий**
  Автоматический расчёт дневной нормы на основе вашего максимума за один подход

* 📝 **Установить норму**
  Ручная установка персональной дневной цели

* 📈 **Мой прогресс**
  История тренировок и выполнение нормы

* 📊 **Статистика**
  Личная статистика + общий рейтинг пользователей

* ⬅️ **Назад**
  Возврат в главное меню

---

## 🧠 Логика расчёта нормы

Дневная норма рассчитывается на основе рекомендаций **ACSM (American College of Sports Medicine)** с учётом максимального количества повторений.

Это позволяет:

* адаптировать нагрузку под пользователя
* поддерживать прогрессивную перегрузку
* избегать перетренированности

## 🏗 Архитектура

Проект построен по принципам чистой архитектуры:

```
backend/
├── main.go           # точка входа
├── handler           # Telegram-обработчики
├── service           # бизнес-логика
├── repository        # работа с БД
├── presenter         # форматирование ответов
├── cache             # in-memory + file cache
├── migrations        # goose миграции
```

### Основные принципы:

* Dependency Injection
* Интерфейсы для мокирования
* Чёткое разделение слоёв
* Тестируемость каждого уровня

## ⚡ Кэширование

Реализован собственный `TodayCache`:

* In-memory хранение текущих данных
* Автосохранение в файл
* Автоматический сброс при смене дня
* Graceful shutdown через context
* Thread-safe (sync.RWMutex)
